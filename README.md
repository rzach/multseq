# MUltseq

MUltseq is a sequent prover for finite-valued logics.

See the [website](https://www.logic.at/multseq).

## Installation

- Install [SWI Prolog](https://www.swi-prolog.org/).
- Download MUltseq
- Test whether MUltseq works.
  ```
  $ cd multseq/examples
  $ swipl
  ?- [ex_classical].
  ?- halt.
  $ pdflatex ex_classical.tex
  ```
